# Pulse Architects Dashboard

A professional business dashboard for Pulse Architects, showcasing key metrics, plan categories, recent activity, revenue tracking, customer stats, and services overview.

## Requirements

* Single HTML file with Tailwind CSS CDN
* Chart.js for charts
* Brand colors from brief: Primary green #07be8a, Sky blue #79be8a, Dark #172023
* Font: Inter or Bricolage Grotesque
* Responsive design
* Create data.json for dynamic data
* Glass-morphism card style like PG Dashboard

## Files

* index.html: Main dashboard
* data.json: Metrics data
* README.md: Documentation

## Notes

* Built using Tailwind CSS and Chart.js
* Uses glass-morphism card style for a modern look
* Fully responsive design for a great user experience
